# Vulnerability Disclosure Policy

This document outlines AskCc's vulnerability disclosure policy. For more information
about AskCc's approach to security, please visit [askcc.com.cn](https://askcc.com.cn).

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please email us at [support@pyznai.com](mailto:support@pyznai.com)
with details and steps to reproduce.

We commit to acknowledging vulnerability reports immediately, and will work to fix
active vulnerabilities as soon as we can. Critical incidents will be communicated
via email to all affected users.

We appreciate your help in making AskCc more secure for everyone.
Thank you for your support and responsible disclosure.
